# Coach Isi - Inner Circle Pioniergruppe

Eine professionelle Landing Page mit integriertem Admin Dashboard für die Verwaltung von Wartelisten-Einträgen.

## Features

### Landing Page
- Responsive Design mit modernem UI
- Wartelisten-Anmeldung
- Testimonials mit Carousel
- FAQ-Sektion
- Social Media Integration
- Mobile-optimiert

### Admin Dashboard
- Sichere Anmeldung
- Übersichtliche Statistiken
- Erweiterte Suchfunktionen
- Status-Management für Einträge
- Bulk-Aktionen
- CSV-Export
- Responsive Design

## Admin-Zugang

Um das Admin Dashboard zu erreichen, fügen Sie `?admin=true` zur URL hinzu:
```
https://ihre-domain.com/?admin=true
```

## Installation

1. Repository klonen
2. Dependencies installieren:
   ```bash
   npm install
   ```
3. Entwicklungsserver starten:
   ```bash
   npm run dev
   ```

## Deployment

Das Projekt ist für Netlify optimiert und kann direkt deployed werden.

## Technologien

- React 18
- TypeScript
- Tailwind CSS
- Lucide React Icons
- Vite

## Sicherheitshinweise

⚠️ **Wichtig für Produktion:**
- Implementieren Sie eine sichere Backend-Authentifizierung
- Verwenden Sie HTTPS
